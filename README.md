# FutPlot

A full-stack football analytics and visualization platform built with Next.js
## Features

- **Interactive Data Visualization** - Powered by Nivo charts
- **PostgreSQL Database** - Robust data storage and querying
- **Automated Scraping** - Weekly data updates via GitHub Actions
- **Next.js App Router** - Fast, server-side rendering

## Tech Stack

### Frontend
- **Next.js 15** with App Router
- **TypeScript** for type safety
- **React** for icons

### Backend
- **Next.js API Routes**
- **PostgreSQL** database
- **pg** (node-postgres) for database connection
